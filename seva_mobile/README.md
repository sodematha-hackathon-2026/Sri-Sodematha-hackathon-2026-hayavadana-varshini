# Seva Platform - Mobile Application

This is the mobile application for Seva, built with React Native and Expo.

## Structure
- `android/`: Android native project.
- `ios/`: iOS native project.
- `src/components`: Reusable UI components.
- `src/screens`: App screens.
- `src/navigation`: Navigation setup.
- `src/services`: API services.
- `src/store`: State management (Zustand).
- `src/hooks`: Custom hooks.
- `src/utils`: Utilities.
- `src/assets`: Images, fonts, and icons.

## Setup
1. Prerequisites: Node.js 18+, Expo Go app or emulator.
2. Install dependencies: `npm install`
3. Start Expo: `npx expo start`
4. Run on Android: Press `a` in the terminal.
5. Run on iOS: Press `i` in the terminal.
